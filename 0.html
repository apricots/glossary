Glossary

HTML - HyperText Markup Language
  • non-functional
  • compiled by the browser
  • markup uses elements enclosed by brackets
CSS - Cascading Style Sheets
  • changes the way HTML elements look
  • can be linked using in-line, external file, or in the HTML head
  • the more specific the CSS selector, the more priority it is given
JavaScript
  • handles certains events on a website and can make a site interactive
  • different libraries including jQuery, Angular
  • can use script tags at the bottom of an html page to load asynchronously after the rest of page contents have loaded
